# Traffic Spike Runbook

## Symptoms

- Concurrent viewers increase rapidly
- CPU utilization rises
- CDN bandwidth saturation
- Increased startup time

## Possible Causes

- Live sports event
- Viral content
- Marketing campaign
- Unexpected audience growth

## Recommended Actions

1. Enable autoscaling.
2. Increase CDN bandwidth.
3. Increase origin replicas.
4. Enable adaptive bitrate streaming.
5. Monitor scaling metrics.

## Verification

CPU utilization remains below 70%.
